# Advanced Differential Robot Package (ex05)

## Концепция Робота

Робот имеет уникальный дифференциальный дизайн с **расширенной прямоугольной базой**, **высокой стойкой**, **двумя декоративными манипуляторами** и чёрным **информационным экраном**.

## Программа Уникальных Движений: Пятиконечная Звезда

Робот выполняет **автономный повторяемый цикл движения**, чертя точную **Пятиконечную Звезду**. Траектория состоит из 5 прямых лучей и 5 точных поворотов на месте.

Движение управляется **10-фазной машиной состояний** (постоянная линейная скорость $V = 0.3 \text{ м/с}$):

1.  **Нечетные Фазы (1, 3, 5, 7, 9): Движение по Лучу.** Робот движется прямо в течение $\mathbf{6.0 \text{ секунд}}$, формируя луч длиной $\mathbf{1.8 \text{ метра}}$.
2.  **Четные Фазы (2, 4, 6, 8, 10): Поворот на месте.** Робот выполняет резкий поворот на $\mathbf{144^\circ}$ (на внешний угол звезды) в течение $\mathbf{5.0 \text{ секунд}}$, готовясь к следующему лучу.

Этот 10-фазный цикл постоянно повторяется, обеспечивая целостное и автономное черчение фигуры звезды.
